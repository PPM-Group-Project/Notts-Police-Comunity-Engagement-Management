Detailed Description

Database model for storing events that need to be scheduled.

Events to be scheduled are added to this model automatically by calculating recommended date obtained to community model. NOTE: THESE ARE NOT SCHEDULED EVENTS! Each event needs manual review by Events Managers, after review, evented is added to ScheduledEvent models, and deleted from this one.
